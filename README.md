AI Workplace Productivity Hub is a modern, responsive web application that helps professionals, students, and teams improve productivity using AI. The platform provides intelligent tools to automate common workplace tasks, save time, and improve efficiency through a clean and intuitive interface.
Features
📧 Smart Email Generator
📝 AI Meeting Notes Summarizer
✅ AI Task Planner
🔍 AI Research Assistant
💬 AI Chat Assistant
🌙 Light/Dark Mode
📊 Productivity Dashboard
🔔 Task Reminders
📁 File Upload & Document Summarization
⭐ Saved Prompts & Chat History
Tools Used
Lovable AI
React
TypeScript
Tailwind CSS
Vite
OpenAI API (or compatible AI API)
Git & GitHub
Setup Instructions
Clone the repository.
Install dependencies.
Add your AI API key to the environment variables.
Start the development server.
Open the application in your browser.
